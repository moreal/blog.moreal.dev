每日 노트 쓰기 (始作)
=====================

그리고 새롭게 쓰는 글에 對해서는 國漢文混用文으로 作成하기로 하였습니다. 이는
漢字에 익숙치 않아 日本語를 어렵게 느끼는 것 같아 점진적으로 漢字에 더
익숙해지기 위해서 해보는 것입니다. 한자를 외워서 쓰는 것은 아니고 눈에 익히기
위함입니다. 만약 “어, 이것도 漢字인데!” 같은 部分이 있다면 부디 알려주세요.

그렇다고 모든 부분을 漢字로 적겠다는 것은 아닙니다. 글 쓰기가 優先이고
後行적으로 바꿀 것입니다. macOS의 경우 opt+enter로 변환이 가능한 데, 이 글을
쓰는 Zed에서 keymap 중 `alt-enter` 를 非活性化하여 韓國語 IME에서 漢字 變換
機能에 入力 이벤트를 넘기도록 할 수 있습니다.

~~~~ json
[
  {
    "bindings": {
      "alt-enter": null,
    },
  },
]
~~~~

이 짧은 글을 적으면서도 이 단어가 이 漢字인가 찾아보면서 했습니다.
